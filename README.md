總管家保險計算器 (精確閏年版)

這是一個專為 總管家包租代管 設計的保險費用試算工具。透過簡單的介面輸入，即可快速計算含閏年邏輯的保費分攤金額。

🚀 線上使用

如果您已開啟 GitHub Pages，可以直接透過瀏覽器存取本工具。

🛠 功能特點

基準日自動判定：自動比對「保險起始日」與「申請起始日」，取較晚者作為計費起點。

動態閏年判斷：根據起始年份自動判定分母為 365 或 366 天。

含頭含尾計算：公式自動補正 +1 天，符合保險實務。

響應式設計：支援手機、平板與電腦操作。

📐 計算公式

應繳費用 = [(截止日 - 基準起始日 + 1) / 當年總天數] × 保險年費
